# DirectDraw2D_Game

EXE 실행 파일과 제가 구현한 소스 코드만 레포지토리에 포함하였습니다.  
소스 코드에 대한 설명은 주석으로 추가해 두었습니다.  

**개발 기간** : 3주  
**팀 구성** : 2인팀

[유튜브 영상](https://youtu.be/iQB3RYAKHIA?si=Pz4x_eIdLN7srZDO)

<br>

## 구현 목록

- **기본 시스템**
  - DebugManager
  - InputManager
  - SceneManager (비동기 Scene 전환) 
  - 카메라

- **게임 오브젝트**
  - 캐릭터 & 몬스터
  - Sprite 애니메이션


- **충돌 처리**
  - Raycast
  - AABB 충돌 검출
  - OBB 충돌 검출
  - Collision Layer
  - 충돌 반응
  - 간단한 2D 물리 (중력 및 충돌 시 밀어내기)


- **기타**
  - 기존 Game클래스 구조 리팩토링
  - UI
